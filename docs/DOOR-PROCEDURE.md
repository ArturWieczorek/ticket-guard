# Door procedure / Procedura przy wejściu

A one-page guide for staff at the door. Print it and keep it at the desk.
Jednostronicowa instrukcja dla obsługi przy wejściu. Wydrukuj i trzymaj przy biurku.

---

## What the scanner colours mean / Co oznaczają kolory skanera

| Screen shows / Na ekranie                                       | Meaning / Znaczenie                                              | Do this / Zrób to                                      |
| --------------------------------------------------------------- | ---------------------------------------------------------------- | ------------------------------------------------------ |
| 🟢 **Ważny - zapraszamy!** / **Valid - welcome!**               | Ticket is good, first use. Bilet dobry, pierwsze użycie.         | **Admit. Wpuść.**                                      |
| 🔴 **JUŻ WYKORZYSTANY** / **ALREADY USED** (with a time)        | This ticket was already scanned in. Ten bilet już zeskanowano.   | **Do not admit. Nie wpuszczaj.** (See below.)          |
| 🔴 **To nie jest kod TicketGuard** / **Not a TicketGuard code** | Not our QR / damaged. To nie nasz kod / uszkodzony.              | **Do not admit. Nie wpuszczaj.**                       |
| 🔴 **Niewłaściwe wydarzenie** / **Wrong event**                 | Ticket is for a different event. Bilet na inne wydarzenie.       | **Do not admit. Nie wpuszczaj.**                       |
| 🔴 **Nie można zweryfikować** / **Can't verify**                | No internet - can't check. Brak internetu - nie można sprawdzić. | **Use the paper list** (below). Użyj listy papierowej. |

---

## Already used - what to do / Już wykorzystany - co robić

One ticket admits one person. The first scan wins; every later scan of the same
ticket is refused. You do **not** need to find who used it first - tickets are
anonymous, and a shared/copied ticket means the people cooperated.

Jeden bilet = jedna osoba. Liczy się pierwsze skanowanie; każde kolejne tego
samego biletu jest odrzucane. **Nie** musisz szukać, kto użył go pierwszy -
bilety są anonimowe.

1. Say politely: "This ticket was already scanned in at (time)."
   Powiedz uprzejmie: „Ten bilet został już zeskanowany o (godzina)."
2. Do not let them in on that ticket. Nie wpuszczaj na ten bilet.
3. If they dispute it, call a supervisor. Jeśli protestują - poproś przełożonego.

---

## Emergency: no internet / Awaria: brak internetu

If the scanner shows **"Can't verify"**, the network is down. Do not guess.
Jeśli skaner pokazuje **„Nie można zweryfikować"**, sieć nie działa. Nie zgaduj.

1. Switch to the **printed backup list** for the event. Przejdź na **wydrukowaną
   listę zapasową** dla wydarzenia.
2. Match **both** the ticket number **and** the code on the paper ticket.
   Sprawdź **numer** biletu **oraz** kod na papierowym bilecie.
3. Cross the number off the list once admitted. Skreśl numer po wpuszczeniu.

> Print the backup list (Generate tab → "Download emergency backup list") and the
> tickets **before** the event, while you still have internet.
> Wydrukuj listę zapasową i bilety **przed** wydarzeniem, póki jest internet.
